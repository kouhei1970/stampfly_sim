# StampFly Drone Simulator

## Dependency

### VPython
```conda install -c vpython vpython```
 or
 ```conda install -c conda-forge vpython```
 or
```pip install vpython```

### HID
```pip install hid```

### Numpy
```pip install numpy```

### Matplotlib
```pip install matplotlib```

### PlatformIO


### ATOM JoyStick 

## Install and Run
```
mkdir github
cd github
git clone https://github.com/kouhei1970/stampfly_sim.git
cd stampfly_sim
python test_sim
```
